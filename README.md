# 待办事项清单

一个简洁、响应式的待办事项 Web 应用，使用原生 HTML、CSS 和 JavaScript 构建。所有数据通过浏览器的 `localStorage` 本地存储，无需后端，无需安装任何依赖。

## 功能特性

- **添加任务**，支持三种优先级：高、中、低
- **行内编辑任务**，无需跳转页面
- **勾选完成**，一键标记任务状态
- **删除任务**，支持单条删除
- **批量清空**，一键清除所有已完成任务或全部任务
- **统计面板**，实时显示已完成和未完成的任务数量
- **自动排序**，按优先级从高到低排列
- **本地持久化**，基于 `localStorage`，刷新或关闭浏览器不丢数据
- **响应式布局**，适配桌面端、平板和手机

## 快速开始

克隆仓库，用浏览器打开 `todo.html` 即可使用：

```bash
git clone https://github.com/YOUR_USERNAME/todo-list.git
cd todo-list
open todo.html   # 或直接双击 todo.html
```

无需构建工具、包管理器或服务器。

## 项目结构

```
├── todo.html               # 主应用文件（HTML + CSS + JS）
└── logistic_regression.py  # 基于逻辑回归的手写数字识别
```

## 使用说明

1. 在输入框中输入任务内容
2. 选择优先级（高 / 中 / 低）
3. 点击 **添加任务** 按钮（或按回车键）
4. 勾选复选框标记已完成，点击 **编辑** 或 **删除** 按钮进行修改
5. 使用 **清空已完成** 移除所有已完成任务，或 **清空所有** 重新开始

## 技术栈

- **HTML5** — 语义化结构
- **CSS3** — 渐变背景、过渡动画、Flexbox 响应式布局
- **JavaScript (ES6+)** — DOM 操作、事件处理、localStorage API
